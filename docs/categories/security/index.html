  • JWT

    2019-12-31

    A JSON Web Token (JWT) is a type of authentication token used to identify a user to a server application. JWTs contain information about the client caller, and can be used as part of a client session architecture. A JSON Web Key Set (JWKS) contains the cryptographic keys used to verify incoming JWTs. You can use Istio’s Authentication API to configure JWT policies for your services.
